
# git.ig

**Git.ig** is a npm package which allows you to quickly create a .gitignore file without the hastle of writing it.


## Installation

Install my-project with npm

```bash
  npm i @aryaanish/git.ig -g
```
    
## Usage/Examples

```bash
git.ig
```

```bash
or

git.ig folder/
```
